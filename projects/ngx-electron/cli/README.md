# @ngx-electron/cli


## Install

```
npm i @ngx-electron/cli -g
```

## Usage

```
ngx-electron new test
```

options 
* --store 
* --skipInstall


## Equivalence

```
npm i @angular/cli@angularCliVersion

ng new test

cd test

ng add @ngx-electron/schematics

or 

ng add @ngx-electron/schematics --store --skipInstall
```
